# Sistema-L-gico-de-Orienta-o-Profissional-Baseado-em-Compet-ncias-Futuras
O texto aborda o desafio atual de escolher uma profissão devido à multiplicidade de capacidades e talentos das pessoas. Para solucionar isso, foi criado um sistema que utiliza essa mesma lógica para identificar e valorizar os pontos fortes do indivíduo, oferecendo-lhe a resposta profissional de que precisa.
